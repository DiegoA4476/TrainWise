# TrainWise

TrainWise is a fitness application inspired by [TheFork](https://www.thefork.com/) — but built for gyms. It combines gym discovery, workout tracking, and AI-powered training insights in a single platform.

---

## Overview

TrainWise allows users to:

- Discover and explore gyms  
- Store and manage custom workouts  
- Receive AI-generated workout overviews  
- Get AI-assisted workout planning  
- Track personal fitness data (e.g., BPM and other metrics)  
- Potentially use camera-based repetition counting for selected exercises  

The goal is to create a smart, centralized fitness companion that enhances both gym selection and training performance.

---

## Core Features

### Gym Discovery
- Browse and compare gyms  
- View key information and availability  
- Simplified booking-style experience  

### Workout Management
- Create and store custom workouts  
- Organize exercises by category or goal  
- Access workout history  

### AI Integration
- AI overview of logged workouts  
- Intelligent workout planning suggestions  
- Data-driven training feedback  

### Personal Metrics Tracking
- Heart rate (BPM)  
- Additional personal performance data  
- Progress monitoring over time  

### Camera-Assisted Repetition Counting (Planned)
- Exercise repetition detection via device camera  
- Automated counting for selected movements
